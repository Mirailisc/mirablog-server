# mirablog-server
## Setup
Install dependencies:
```
npm install
```
or
```
yarn install
```

Setup `.env` file:
```
DATABASE_URL="mongodb+srv://admin_mirailisc:Miracle_48@cluster0.qjrht.mongodb.net/Test"
```
for development only
```
PORT=4000
```
